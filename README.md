# Online Store Management System

This project is a web-based system for managing an online marketplace, designed to facilitate the buying and selling of goods between users in distinct roles: buyers and vendors.
It includes detailed requirements analysis, UML modeling, database design, and a Java/Spring implementation with Hibernate persistence.

## Technical Highlights

- **Backend**: Java with Spring Framework
- **Persistence**: Hibernate ORM with JPA annotations
- **Validation**: Jakarta Bean Validation
- **Inheritance modeling**:
  - Overlapping roles via role fields
  - Employee type transitions via dynamic fields and methods
- **Associations**:
  - One-to-one, one-to-many, and many-to-many
  - Composition relationships for richer domain modeling
- **Attribute modeling**:
  - Optional and multi-value attributes with collections
  - Derived attributes via dynamic getters
  - Tagged values via enum types
